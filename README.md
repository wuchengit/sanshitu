# 三视图生成器 (Sanshitu)

AI 人物三视图生成工具，前端 SPA + PHP 后端。

## 功能特性

### 人物资产结构（7 种）
- **经典三视图** — 9:16 面部+三视图
- **全身定妆照** — 9:16 正面站立
- **纯三视图** — 16:9 全身+头部
- **面部三视图** — 16:9 面部三视角
- **三视图+细节** — 16:9 三视图+特写
- **半身三视图** — 16:9 半身三视角
- **头像三视图** — 全身照+头部三视图

### 人物风格（18 种）
通用、电影角色、冷峻硬朗、奇幻精灵、赛博朋克、战士、古风雅韵、暗黑Noir、甜美清新、尊贵气场、街头潮流、梦幻仙境、科技未来、废土末日、暗黑哥特、日系校园、维多利亚、极简禅意

### 艺术风格（12 种）
无、3D CG、超写实、动漫、剪纸、布偶、油画、水彩、素描、像素风、黏土、水墨

### 导演代表作（11 位）
无、王家卫、北野武、斯皮尔伯格、宫崎骏、张艺谋、诺兰、韦斯·安德森、昆汀、库布里克、蒂姆·伯顿、黑泽明

### 人物属性
性别（男/女）、人种（7 种）、年龄（小孩/年轻/成熟/中年/老者）、身高（高/中/娇小）

### 职业选项（30 种）
战士、法师、刺客、猎人、牧师、骑士、武士、忍者、海盗、牛仔、侦探、医生、士兵、飞行员、厨师、画家、音乐家、科学家、宇航员、探险家、武僧、吸血鬼、狼人、精灵射手、矮人、机器人、天使、恶魔、龙骑士、机甲驾驶员

### 影视年代（20 种）
远古石器→古埃及→古希腊→古罗马→中世纪→文艺复兴→维京→日本战国→大唐→武侠→维多利亚→西部→1920s上海→二战→1950s美国→1960s嬉皮→1980s复古→赛博时代→末日废土→太空科幻

### 其他功能
- **多语言支持** — 6 种语言（中/英/日/韩/法/西）
- **用户认证** — Logto OIDC 登录，token 存储在 localStorage
- **自定义预设** — 多维度自定义预设系统，支持保存/另存/编辑
- **配置云端存储** — 登录后可云端同步用户配置（AES-256-CBC 加密）
- **参考图上传** — 最多 6 张参考图
- **浮层图片查看器** — 滚轮缩放 + 拖动 + 下载 + 信息显示
- **翻译功能** — DeepSeek API 翻译提示词，结果缓存
- **生成历史** — localStorage 存储，30 条上限，支持多选/批量下载/删除
- **生成历史** — 瀑布流展示，缩略图 JPG 格式，支持批量下载 zip

## 技术栈
- 前端: HTML/CSS/JS (单文件 SPA)
- 前端(Vue): sanshitu-app/ (Vue 3 + Vite，开发中)
- 后端: PHP 8.3
- 图片生成: grsai API (gpt-image-2 / gpt-image-2-vip)
- 翻译: DeepSeek API
- 认证: Logto OIDC
- Web 服务器: Nginx

## 认证说明
- 登录走 Logto OIDC → `callback.html` 处理回调 → token 存入 localStorage
- `api/user/*.php` 目前使用旧 JWT 认证（`api/lib/auth.php`）
- 需迁移：校验 Logto access_token 替代旧 JWT
- 迁移方案：已在 `api/lib/logto_auth.php` 创建，解析 Authorization header 中的 Bearer token，向 auth.aiwuuw.com/oidc/me 验证
